## Où trouver les icônes ?

Tu peux :
1. Les extraire des assets du jeu
2. Les télécharger depuis le CDN d'Ubisoft
3. Les trouver sur des sites de ressources R6 (r6tracker, r6stats, etc.)

## Format requis

- Format : PNG
- Taille recommandée : 500x500px
- Fond transparent de préférence

## Nomenclature

Les fichiers doivent suivre ce format exact :
- `R6S_RANK_500x500_{RankName}_{Division}.png`
- Division : `01` = I, `02` = II, `03` = III, `04` = IV, `05` = V

Exemple :
- "Emerald I" → `R6S_RANK_500x500_Emerald_01.png`
- "Diamond V" → `R6S_RANK_500x500_Diamond_05.png`
